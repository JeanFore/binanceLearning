# Binance Learning Prompt Guide

Esta guia contiene comandos y prompts para usar con el GPT de Binance Learning mientras aprendes arbitraje cripto en modo educativo.

Usa estos prompts tal cual o adaptalos a tu situacion.

## Inicio de Cada Chat

Usa este prompt al iniciar un chat nuevo:

```text
Consulta mi contexto de aprendizaje y dime que debo estudiar ahora.
```

Si quieres un resumen mas completo:

```text
Consulta mi contexto, resume mi avance, dime que clases tengo pendientes y recomiendame el siguiente paso.
```

Si vienes de una sesion anterior:

```text
Consulta mi contexto y mis notas recientes. Retomemos desde donde quede.
```

## Elegir Que Estudiar

```text
Dime cual es la siguiente clase recomendada segun mi progreso.
```

```text
Revisa mi avance y proponme una ruta de estudio para hoy de 30 minutos.
```

```text
Quiero estudiar una clase teorica antes de hacer practicas. Que sigue?
```

```text
Quiero hacer una practica simulada. Busca una clase adecuada segun mi progreso.
```

## Pedir Explicaciones

```text
Explicame este tema como principiante: [tema].
```

```text
Explicame [tema] con un ejemplo numerico simple y sin usar dinero real.
```

```text
Explicame por que [tema] importa para arbitraje.
```

```text
Comparame estos conceptos: [concepto A] vs [concepto B].
```

```text
Explicame [tema] y luego dame una practica simulada.
```

## Cuando Algo No Este En El Contenido

```text
Revisa si este tema ya existe en el contenido. Si no existe, crea un borrador educativo para cubrirlo.
```

```text
Si esta pregunta no esta cubierta por las clases actuales, crea una nueva clase sobre el tema.
```

```text
Convierte esta duda en contenido nuevo del curso y guardalo como borrador: [duda].
```

```text
Si este tema merece una clase formal, crea la clase. Si solo es una explicacion adicional, crea un borrador.
```

```text
Guarda una nota explicando que este tema faltaba y que contenido creaste.
```

## Guardar Avances

Cuando termines una clase:

```text
Termine esta clase. Guarda mi avance como completado y resume lo que aprendi.
```

Cuando entendiste una parte:

```text
Guarda mi avance: ya entendi [concepto]. Todavia quiero practicar mas antes de completar la clase.
```

Cuando hiciste una practica:

```text
Guarda esta practica como evidencia: [describe lo que hiciste].
```

Cuando respondiste una evaluacion:

```text
Evalua mi respuesta, asigna un score si aplica y guarda el progreso.
```

Cuando quieres pasar a la siguiente clase:

```text
Si mi respuesta es suficiente, guarda la clase como completada y dime cual sigue.
```

## Manejar Bloqueos

Si no entiendes algo:

```text
Estoy bloqueado con [tema]. Explicamelo desde cero y guarda el progreso como bloqueado.
```

```text
No entiendo esta parte: [parte]. Crea una explicacion mas simple y una practica corta.
```

```text
Antes de avanzar, evaluame con preguntas sobre [tema].
```

```text
Detecta que prerequisitos me faltan para entender [tema].
```

```text
Crea una practica simulada para calcular net edge.
```

```text
Crea un ejercicio de bid, ask, spread y fees con numeros ficticios.
```

```text
Crea una practica de order book depth sin usar dinero real.
```

```text
Crea una practica de arbitraje triangular solo en spreadsheet o simulacion.
```

```text
Revisa mi calculo y dime si el net edge sigue siendo positivo despues de costos.
```


```text

```


```text

```

```text

```

## Crear Contenido Nuevo

Para una explicacion adicional:

```text
Crea un borrador de contenido complementario sobre [tema].
```

Para una clase formal:

```text
Crea una nueva clase sobre [tema] con objetivo, teoria, practica simulada y criterios de exito.
```

Para una practica:

```text
Crea una clase practica sobre [tema] con datos ficticios, pasos y criterios de evaluacion.
```

Para un modulo:

```text
Crea un modulo nuevo sobre [tema] y agrega una clase teorica y una practica simulada.
```

Para mejorar contenido existente:

```text
Revisa la clase actual y mejora el contenido para que sea mas claro para principiantes.
```

## Preguntas de Seguridad

```text
Antes de cualquier practica, dame el checklist de seguridad.
```

```text
Revisa si esta idea rompe alguna regla de seguridad: [idea].
```

```text
Explicame que riesgos faltan en este plan: [plan].
```

```text
Convierte este plan en una simulacion segura, sin dinero real.
```

```text
Rechaza esta idea si implica trading real, leverage o recomendaciones de tokens.
```

## Prompts Para Estrategias Educativas

```text
Ayudame a disenar una estrategia educativa de arbitraje triangular solo para simulacion.
```

```text
Revisa esta estrategia como si fueras auditor de riesgo: [estrategia].
```

```text
Identifica supuestos irreales en este plan: [plan].
```

```text
Agrega reglas de stop, timeout, slippage maximo y journaling a esta simulacion.
```

```text
No apruebes la estrategia si no incluye fees, slippage, liquidez y escenario de fallo.
```

## Prompt Maestro Para Usar Siempre

Puedes pegar este prompt al inicio de una sesion larga:

```text
Actua como mi tutor de Binance Learning. Primero consulta mi contexto. Luego dime donde estoy, que deberia estudiar ahora y proponme una actividad segura en modo simulacion. Si durante la conversacion detectas que falta contenido, crea un borrador o una clase nueva. Si completo una actividad, guarda mi progreso. Si tengo una duda importante, guarda una nota para retomarla despues. No des asesoría financiera ni recomendaciones de tokens o trading real.
```

## Regla Final

Si quieres que algo quede guardado, dilo explicitamente:

```text
Guarda esto.
```

```text
Guarda mi avance.
```

```text
Guarda una nota.
```

```text
Crea contenido nuevo y guardalo.
```

El GPT debe usar las Actions para guardar la informacion en el source.
